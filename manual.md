# Cream by Solstar マニュアル

Shopifyテーマ「Cream by Solstar」を最短で使いこなすための実践ガイドです。各セクションが「何をするか」「どこを触れば形になるか」「迷ったときのヒント」がすぐ分かるようにまとめています。

## まずはここから（基本操作）
- 管理画面: 「オンラインストア > テーマ > カスタマイズ」から編集します。
- テンプレート切替: 上部プルダウンでホーム/商品/コレクション/ブログ/検索/カートなどを選択。
- セクション編集: 左サイドバーでセクション/ブロックを追加・並べ替え・非表示・削除。
- メディア管理: 画像/動画は「設定 > ファイル」にアップロードして選択。
- 反映確認: 保存後にプレビュー/公開で確認。

## セクション共通の見た目調整
- ここを触る: 各セクションの「設定」パネル。
- よく使う項目: 幅（ページ幅/全幅）、カラースキーム、上下余白、ブロック間隔、テキスト配置。
- ヒント: カラースキームは「テーマ設定 > 色」で事前に作っておくと、複数セクションで統一感を出しやすくなります。

## テーマ設定（全ページに効く項目）

### ロゴとファビコン
- 役割: ヘッダー/フッターのロゴとサイトアイコン。
- 主な設定: デフォルト/逆さロゴ、ロゴ高さ(PC/モバイル)、ファビコン。
- ヒント: 暗い背景用に逆さロゴを用意すると配色切替時に便利。

### 色
- 役割: 背景・文字・ボタンなどの配色セットを作成。
- 主な設定: 背景、見出し/本文、プライマリ、罫線/影、ボタン色。
- ヒント: セクション別に配色を変えたい場合も、ここでスキームを複数用意しておく。

### タイポグラフィー
- 役割: 本文/見出し/アクセントのフォントとサイズ。
- 主な設定: 見出し1-6、本文、行間、字間、ケース。
- ヒント: 見出しと本文の対比を強めるとカードやヒーローの訴求力が上がります。

### ページレイアウト
- 役割: ページ幅とパンくず表示の有無。
- ヒント: サイト幅を狭めると商品カードが縦長に、広げると横にゆとりが出ます。

### アニメーション
- 役割: ページ遷移やホバー動作を一括でオン/オフ。
- ヒント: 初期構築中はオフ、公開前にオンにするとプレビューが軽くなります。

### バッジ
- 役割: 商品カードのSale/Sold out/New/Bestsellerバッジ。
- 主な設定: 表示位置、ラベル、色、角丸、割引表示（％/金額）、Bestsellerタグ名。

### ボタン
- 役割: プライマリ/セカンダリ/ピルボタンの形と書体。
- 主な設定: 太さ、角丸、フォント、テキストの大文字/小文字。

### カート
- 役割: カートの挙動と表示形式。
- 主な設定: ドロワー自動表示、メモ欄、割引欄、分割払い表示、クイックチェックアウト、空カートのリンク。

### ドロワー
- 役割: カートや検索ドロワーの見た目。
- 主な設定: 配色、罫線、影。

### アイコン
- 役割: アイコンの線幅を一括調整。

### 入力フィールド
- 役割: フォーム入力欄の枠線/角丸/文字スタイル。

### ポップオーバーとモーダル
- 役割: ポップオーバー/モーダルの配色・影・角丸。

### 価格
- 役割: 価格表記と通貨コードの表示スタイル。
- 主な設定: 商品ページ/カード/カートの表示切替。

### 商品カード
- 役割: 商品カードの表示挙動。
- 主な設定: クイック追加、2枚目画像ホバー、カルーセル表示。

### お気に入りリスト
- 役割: ウィッシュリストの色とアイコン。

### ビデオイントロ
- 役割: 初回訪問時のイントロ動画。
- 主な設定: 有効化、ホームのみ表示、自動再生、音量、デスクトップ/モバイル動画、スキップボタンの文言とスタイル。

### 検索
- 役割: 検索ポップオーバーと空状態の見た目。
- 主な設定: 空コレクション表示、角丸、タイトルの大文字/小文字。

### 見本（スウォッチ）
- 役割: バリエーションスウォッチのサイズと枠。
- 主な設定: 幅/高さ、角丸、罫線。

### バリエーションピッカー
- 役割: バリエーション選択ボタンのスタイル。
- 主な設定: 角丸、罫線、幅の伸縮。

### スプラッシュスクリーン
- 役割: ロード時のロゴ表示。
- 主な設定: 有効化、ロゴサイズ、背景色。

## セクション一覧（カテゴリ別ガイド）

### ヘッダーとナビゲーション

#### お知らせバー
- 表示: 画面上部でメッセージをスライド表示。
- 主な設定: 切替間隔、幅、配色、区切り線。
- 使いどころ: セール告知や配送案内を短文でループ表示。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| 告知テキスト（ブロック） | Text | 1行メッセージ。複数追加でローテーション。 |
| 切替間隔 | Range | メッセージの自動切替速度。 |
| セクション幅 | Select | ページ幅/全幅。 |
| カラースキーム | Color scheme | 背景・文字色セット。 |
| 区切り線 | Checkbox/Range | 上下の境界線有無と太さ。 |

#### ヘッダー
- 表示: ロゴ、メニュー、検索、アカウント、ローカライズ。
- 主な設定: ロゴ位置、メニュー行構成、検索アイコン位置、国/言語セレクター、配色/余白/境界線。
- ヒント: グローバルナビを2段にするとカテゴリ数が多いショップでも崩れにくい。
- 使う場面: 常時表示のグローバルナビで主要導線を整理するとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| レイアウト（位置/行） | Select | ロゴ・メニュー・アイコンの並びを選択。 |
| メニュー | Link list | メイン/サブのメニュー割り当て。 |
| 検索アイコン | Checkbox | ヘッダーに検索トリガーを表示。 |
| アカウント/ローカライズ | Checkbox | アカウントリンク、国/言語セレクターの表示。 |
| カラースキーム/境界線 | Color scheme/Range | 配色と上下ボーダーの有無・太さ。 |
| パディング | Range(px) | ヘッダー上下余白。 |

#### 追従メニュー
- 表示: モバイル下部の固定ボトムメニュー。
- 主な設定: アイコン/ラベル、色、サイズ。
- ブロック: カスタム、ホーム、カート、お気に入り、LINE、アカウント、検索、カテゴリー、もっと見る。
- 使いどころ: モバイルで主要導線を常に表示したいとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| カラースキーム | Color scheme | 背景・文字色セット。 |
| アイコンサイズ/ストローク | Range | ボタンアイコンの大きさと線幅。 |
| 高さ/角丸 | Range(px) | バー全体の高さと角の丸み。 |
| ブロック: 各リンク | Preset/URL | 目的地（ホーム/カート等）または任意URLを設定。 |
| ラベル表示 | Checkbox | アイコンの下にテキストを表示するか。 |

#### 検索（ページ上部）
- 表示: 検索ページの検索UI。
- 主な設定: 配置、配色、余白。
- 使う場面: 検索結果ページの先頭に検索バーを再掲し、再検索しやすくするとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| 配置 | Text alignment | 入力フォームとボタンの寄せ位置。 |
| カラースキーム | Color scheme | 背景・文字色セット。 |
| 上下余白 | Range(px) | セクション外側の余白。 |

#### 予測検索
- 表示: 入力中の候補リスト。
- 主な設定: テーマ設定 > 検索で外観調整。
- 使う場面: ヘッダー検索から即座に候補を出して離脱を防ぎたいとき。

設定詳細
| 設定 | 種別 | 説明 |

| カラースキーム | Color scheme | ドロワー/ポップオーバーの配色。 |
| 角丸/影 | Range/Checkbox | コンテナの角丸と影。 |
| 空状態文言 | Text | ヒットが無い場合のメッセージ。 |

#### 予測検索なし
- 表示: 予測検索でヒットが無い場合の空状態。
- 主な設定: テーマ設定 > 検索の空状態スタイル。
- 使う場面: 候補ゼロ時に人気コレクションや特集への導線を提示したいとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| タイトル/説明 | Text | 空状態の案内文。 |
| ボタンラベル/リンク | Text/URL | 次の行動を促す導線。 |
| カラースキーム | Color scheme | 背景・文字色セット。 |

### フッター

#### フッター
- 表示: サイト下部のリンク/フォーム/決済アイコンなど。
- 主な設定: 配列、配色、余白。
- ブロック例: メニュー、SNS、メール登録、ロゴ、テキスト、ボタン、決済アイコン。
- 使う場面: 会社情報・サポート・SNS・決済手段をまとめ、信頼性と回遊導線を補強するとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| レイアウト/列数 | Select/Range | ブロックの並べ方とカラム数。 |
| カラースキーム | Color scheme | 背景・文字色セット。 |
| 境界線/余白 | Range(px) | 上部ボーダーと内外の余白。 |
| ブロック（メニュー/SNS等） | Preset | 各ブロックでリンクやアイコンを設定。 |

#### ユーティリティ
- 表示: 最下段のコピーライト/ポリシー/ソーシャル。
- 主な設定: 幅、隙間、区切り線、配色、余白。
- 使う場面: 法務リンクや著作権表示を整理し、最低限の導線だけをコンパクトに残すとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| セクション幅 | Select | ページ幅/全幅。 |
| カラースキーム | Color scheme | 背景・文字色セット。 |
| 区切り線 | Checkbox/Range | 上境界線の有無と太さ。 |
| ブロック: コピーライト/ポリシー/SNS | Text/Link list | 表示文言とリンク先。 |
| 上下余白 | Range(px) | セクション外側の余白。 |

| 配置 | Select | 左/中央/右寄せ。 |
| カラースキーム | Color scheme | 背景・文字色セット。 |

#### パスワードフッター
- 表示: パスワードページ用フッター。
- 主な設定: 配色。
- 使う場面: クローズド期間のランディングでブランドらしさを最低限示すとき。

設定詳細
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| カラースキーム | Color scheme | 背景・文字色セット。 |

- 使う場面: ストア公開前や限定公開時に雰囲気を保ちつつ案内したいとき。

## 連携・注意事項
- Search & Discoveryアプリでフィルターや補完商品を設定できます。
- 商品レビュー表示には別途レビューアプリが必要です。

## Tips（運用のコツ）
- 画像/動画: ヒーローは横長16:9か4:5、高さに迷ったら70vh。WebP優先、動画は短尺mp4(H.264)、モバイルは軽量版を用意。
- 配色とロゴ: カラースキームごとにロゴ（通常/逆さ）を確認。暗い背景では逆さロゴを使う。
- モバイル最適化: 列数を減らし余白を詰める。追従メニューを有効化し、ヒーローはモバイル画像を別設定。
- パフォーマンス: 不要なポップアップやオートプレイをオフ。画像は事前リサイズ、動画は音声オフ＋短尺を前提に。
- アクセシビリティ: 画像代替テキストを入れる。ボタン文言は動詞から。コントラストが低い配色は避ける。
- 多言語/多通貨: ローカライズスイッチを有効にし、言語別ロゴ/コピーを確認。静的テキストは言語別で複製して出し分ける。
- 商品訴求: バリエーションが多い商品はスウォッチサイズを調整。追従カート＋クイック追加を併用するとCVR向上が期待できる。
- 検索/フィルター: Search & Discoveryでフィルター/並び替えを設定し、空状態メッセージを編集して回遊先を提示。
- よくあるつまずき: 地図はAPIキー未設定だと非表示。ニュースレターポップアップはPC/モバイル両方でテスト。動画が重いと再生が遅くなる。

## 公開前チェックリスト
- ナビ/フッター: リンク切れなし、法務リンクとコピーライト更新、ロゴ視認性を各配色で確認。
- 配色: カラースキーム割当と文字コントラストをチェック。
- モバイル: 列数・余白・フォント、追従メニュー、ヒーローのモバイル画像を確認。
- 商品ページ: バリエーション選択、スウォッチ、追従カート、クイック追加、価格/通貨表示を確認。
- 検索/フィルター: Search & Discoveryの設定と空状態メッセージを確認。
- カート/チェックアウト導線: メモ・割引・送料表示、ボタン文言、挙動を確認。
- システムページ: 404とパスワードの文言と導線を差し替え済みか。
- パフォーマンス: 不要セクション/ポップアップを無効化、画像圧縮、動画は短尺に。
