# Cream by Solstar コレクションページマニュアル

コレクション（商品一覧）テンプレート専用のガイドです。ホームページや商品詳細と共通のセクションは対象外なので、必要に応じて各マニュアルも併読してください。

## まずはここから（基本操作）
- テンプレート切替: テーマエディタ上部のプルダウンから「コレクション」を選択し、検証したいコレクションを指定。
- コレクション切替: 右上の「コレクションを変更」で商品点数やタグ構成が近いものを選んで動作を確認。
- ページ固有設定: コレクションテンプレートでは「Main collection」セクションが必須です。構成はギャラリー＋フィルターをこのセクション内で完結させます。
- プレビュー: PC/モバイルの両方でレイアウト切替、フィルタードロワー、無限スクロール、URLに#付きアンカーがある場合のスクロール挙動を確認。

## セクション共通の見た目調整
- ここを触る: セクション右側の設定パネル（幅、余白、カラースキーム、ギャップ）。
- よく使う項目: セクション幅（ページ/全幅）、上下余白、列間/行間、モバイル全幅化、タイトル行数制限。
- ヒント: コレクションはカード数が多くなるため、上下余白は32px以下に抑えるとスクロール量を圧縮できます。背景色を変えるときはフィルターエリアも同じカラースキームになる点に注意。

## テンプレート（コレクション専用セクション）

### Main collection
- 目的: フィルター/ソート/グリッド密度と商品カードの表示をまとめ、コレクションページの中核を構成。
- 含まれる機能: 縦型/ドロワーフィルター、検索&Discovery連携のタグ/価格フィルタ、ソート、表示件数（無限スクロール）、商品カード（ギャラリー/タイトル/価格/スウォッチ）、グリッド幅とギャップ調整。
- 使う場面: すべてのコレクションテンプレートで必ず1つ設置します。上部に「セクション（汎用）」を置いてイントロを足す場合も、このセクションより下に並べます。

設定詳細（セクション）
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| レイアウトタイプ | Select | `grid`（通常）/`organic`（編集スタイル）。Gridのみカードサイズと行間設定が表示されます。 |
| 商品カードサイズ | Select | small/medium/large/extra-large。画像比率とテキストサイズを一括で変更。Grid時のみ。 |
| モバイルカードサイズ | Select | small/large。1列表示の高さと余白を調整。 |
| 無限スクロール | Checkbox | ONでページネーションなしの自動ロード。OFF時は「ページあたり商品数」スライダーが有効。 |
| 1ページ商品数 | Range | 8-36。無限スクロールOFF時のみ表示。 |
| グリッド幅 | Select | `centered`（ページ幅内）/`full-width`（左右端まで）。centered時は「モバイルを全幅にする」をON/OFF可。 |
| モバイル全幅化 | Checkbox | グリッド幅=pageのとき、スマホのみ余白をなくしてカードを広げる。 |
| 横ギャップ/縦ギャップ | Range(px) | カード間のスペース。横は16px、縦は24pxがデフォルト。 |
| タイトル行数制限 | Checkbox | ONで商品名を2行に強制し、長尺タイトルの高さを揃える。 |
| 左右パディング | Range(px) | グリッド全体の左右余白。スライダーで0-100px。 |
| カラースキーム | Color scheme | 背景/文字のセット。フィルター領域も同色になります。 |
| 上下余白 | Range(px) | セクション外周の余白。下部に24-32px確保するとフッターとの差が付きます。 |

ブロック: フィルター&コントロール
- 有効化: 「絞り込みを有効にする」をONにするとSearch & Discoveryで設定したフィルターが表示されます。
- 表示スタイル: `vertical`（PC左側固定）か`horizontal`（ドロワー）。`filter_width`で中央寄せ/左右いっぱいを切替。
- ラベル: ラベルの大文字化、スウォッチラベル表示、フィルタータイトルの表示/非表示を個別に設定可能。
- ソート & グリッド密度: `enable_sorting`でソートセレクター、`enable_grid_density`でカード間隔を1クリックで切替えるUIを表示。
- パディング/マージン: padding-block、facets_marginを使い、フィルター周囲の余白を調整。

ブロック: 商品カード（_product-card）
- ギャラリー: 画像比率（portrait/square/landscape/adapt）、枠線、角丸、余白を設定。`media extend`オプションと合わせて視覚ボリュームを調整。
- タイトル/価格: フォントプリセット、カラー、余白を選択可能。税込み表記やセール価格表示もここで制御。
- スウォッチ: PC/モバイルの整列、パディング有無を設定。スウォッチデザイン自体はテーマ設定 > 見本（スウォッチ）に従います。

Tips
- 無限スクロールON時はアンカーリンク（URLに#）でアクセスしても、対象カードが表示されるまで自動ロードされます。商品数が多い場合、検証用にOFFへ切り替えてスクロール量を把握してください。
- Verticalフィルターを使う場合は、横幅の広いモニターでの余白最適化のため`product_grid_width`をpageにして、`facets_margin_right`を16px以下にするとバランスが取りやすいです。
- Sort/密度スイッチはタップ領域が広いので、モバイルでの誤タップを防ぐには`filter_style`をhorizontalにし、ドロワー内にまとめる構成も有効です。

## 連携・注意事項
- フィルター内容はSearch & Discoveryアプリに依存します。アプリ側でメタフィールドやタグを紐付けていない項目は表示されません。
- 商品カード内のスウォッチ色や表示名は「テーマ設定 > 見本（スウォッチ）」で管理しています。コレクション側での個別上書きはできません。
- `results-list.js`を使用しているため、別セクションで商品カードを大量に追加するとJSが重複実行する可能性があります。コレクションページではなるべく商品カードをMain collection内に集約してください。

## Tips（運用のコツ）
- Hero的なイントロが必要な場合は、Main collectionの上に汎用セクション（例: セクション（汎用））を1つだけ追加し、残りはMain collectionに任せると整理しやすいです。
- 画像比率: アパレルや縦長商品は`portrait`、雑貨やギフトは`square`で揃えるとカード一覧のリズムが整います。
- 検索導線: フィルターを多用する場合はヘッダーの検索やコレクション説明文にも主要タグを書くと離脱を防げます。
- パフォーマンス: Infinite scroll + 大量画像のときはLazy loadが効くものの、GIFや自動再生動画は避けてください。

## 公開前チェックリスト
- フィルター: Search & Discoveryで設定した順序どおりに並んでいるか、重複や空ラベルがないか。
- ソート/密度: すべてのオプションが翻訳済みか、選択状態がURLパラメータに反映されるか確認。
- 商品カード: セール品/在庫切れ/バリエーションありの商品でタイトル・バッジ・スウォッチが崩れないか。
- 無限スクロール: 最下部到達後に自動で次ページが読み込まれるか、読み込み中表示が切れないか。
- SEO: コレクション説明文が必要な場合、上部の汎用セクションではなく`collection.description`ブロックを使ってテキストを出力しているか。
