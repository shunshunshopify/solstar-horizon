# Cream by Solstar コレクションリストマニュアル

ホーム/ランディングで複数コレクションをまとめて紹介する「コレクションリスト」セクション専用のガイドです。カード配置、カルーセル、ベントー（不定形グリッド）などのバリエーションを活用し、カテゴリ導線を素早く整えられるようにまとめました。

## まずはここから（基本操作）
- 追加場所: テーマエディタで任意のテンプレートに「コレクションリスト」を追加。Heroやコピーの直下に置くと効果的。
- 対象コレクション: 「コレクションリスト」設定で表示したいコレクションを複数選択。選択しない場合はレイアウト種別ごとの上限数でプレースホルダーが生成されます。
- レイアウト切替: `grid / carousel / bento / editorial` を切り替えながらPC/モバイルプレビューを確認し、表示数と余白を調整。
- リンク先確認: 各コレクションカードは自動で該当コレクションURLへ遷移します。リンク切替はコレクション自体のURL変更で一括反映。

## セクション共通の見た目調整
- ここを触る: 右サイドバー「セクション設定」。カード数・レイアウト・ギャップ・セクション幅・配色をまとめて設定。
- よく使う項目: `layout_type`、`columns`、`carousel_on_mobile`、`section_width`、`gap`、上下余白。
- ヒント: セクション幅をpage-widthにしておくと、他セクションとのリズムが作りやすく、背景色を切り替えても左右余白がそろいます。全幅で使う場合はブロック側で余白を追加するか、画像自体に余白を含めると窮屈さを防げます。

## テンプレート

### コレクションリスト
- 目的: 複数コレクションを1つのセクションで俯瞰表示し、カテゴリ回遊を促進。
- 含まれる機能: グリッド/カルーセル/ベントー/エディトリアルレイアウト、モバイル専用カルーセル、カスタムヘッダーブロック、カード比率/余白調整、ナビゲーションアイコン。
- 使う場面: ホームやギフト特集で複数カテゴリをタイル状に並べたいとき。新着カテゴリや価格帯をスライダーで紹介するケースにも。

設定詳細（共通）
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| コレクションリスト | Collection list | 表示したいコレクションを複数選択。空の場合はレイアウト種別別の最大数までプレースホルダーが生成。 |
| レイアウトタイプ | Select | `grid / carousel / bento / editorial`。選択内容に応じて後続オプションが変化。 |
| セクション幅 | Select | `page-width / full-width`。Hero直下ではpage、連続するコレクション群ではfullに。 |
| セクションギャップ | Range(px) | セクション内上下の要素間余白（ヘッダーとカード群の距離）。 |
| 配色 | Color scheme | 背景と文字色セット。ブロック内のテキスト/ボタンも継承。 |
| 上下余白 | Range(px) | セクション外周の余白。前後に同系色セクションがある場合は上下16px程度に詰めると一体感が出る。 |

レイアウト別オプション
- **Grid**: `columns`（1-8列）、`mobile_columns`（1or2列）、`columns_gap`（横）、`rows_gap`（縦）、`carousel_on_mobile`（モバイルだけスライダー化）。
- **Carousel**: `columns`はスライド内カード数、`mobile_card_size`で60cqw/44cqwを選んでモバイル幅を調整。ナビゲーションは`icons_style`と`icons_shape`で矢印/シェブロン＋円/角/なしを設定。
- **Bento**: `bento_gap`でタイル間余白を管理。最大4枚のコレクションを不均等グリッドで配置するプリセット。ビジュアル訴求が強いときに有効。
- **Editorial**: `max_collections`（1-16）を手動で指定し、余白広めのカードを段組表示。文章ブロックを組み合わせてストーリー性を出す用途に向く。

ブロックの使い方
- 追加できるブロック: テキスト/画像/アイコン/ボタン/グループ/動画/区切り等。セクション冒頭にヘッダーを作り、`group`ブロック内で見出し＋説明＋ボタンをレイアウトすると管理しやすい。
- カスタムCTA: 例としてボタンブロックを追加し「すべての商品を見る」リンクを設置。セクション幅がpage-widthの場合、ブロックの幅も`fill`にすると左右整列がしやすい。

Tips
- カルーセルの`columns`をPC4/モバイル1にし、`carousel_on_mobile`をOFFにするとPCはスライダー、モバイルは横スクロールカード（60cqw）になるため視認性が高い。
- Bentoレイアウトは最大4枚なので、配置するコレクション画像は縦横比を揃えておくとタイルの連続性が出る。
- Editorialはコレクションを選択しなくても「max collections」分のプレースホルダーで骨組みを確認できるので、撮影素材前に構造を固める際に便利です。
- 画像の代わりにカラー背景だけでカードを作りたい場合は、各コレクションの代表画像を単色に差し替えるか、画像内にテキストを入れてタイル風に仕上げます。

## 連携・注意事項
- 表示順は「コレクションリスト」で選択した順番に従います。Shopify管理画面のコレクションソートは影響しません。
- コレクションに画像が未設定の場合、Shopifyのプレースホルダーが表示されるため、公開前に必ず代表画像を登録してください。
- カルーセルのナビゲーションアイコンはテーマ全体のIconフォントを利用します。独自SVGを使いたい場合は`snippets/resource-list-carousel.liquid`側のカスタマイズが必要です。

## Tips（運用のコツ）
- キャンペーンごとに表示コレクションを差し替える場合は、Metaobjectやタグで共通名を付けたコレクションを複製し、公開/非公開で切り替えると運用がスムーズ。
- グリッドで列数を増やしすぎるとテキストが読みにくくなるため、PCは4-5列、モバイルは2列を上限にすると視認性が保てます。
- カルーセルONのセクションが連続するとナビ矢印が重たく見えるため、背景色や上下余白でリズムを変えるのがおすすめ。

## 公開前チェックリスト
- コレクション選択: 非公開コレクションが混ざっていないか、画像が高解像度で揃っているか。
- レイアウト確認: Grid/Carousel/Bento/Editorialの切替で余白や行数が崩れないか、モバイルプレビューで矢印やドットが押しやすいか。
- ブロック: 見出し・説明・ボタンなど追加ブロックのフォント設定やリンク先を再確認。
- 翻訳: カルーセル矢印ラベルやホバー文字が翻訳テーマ（ja/en）で適切に表示されるか。
- アクセシビリティ: コレクション画像に代替テキストが登録されているか（各コレクションの代表画像編集で設定）。
