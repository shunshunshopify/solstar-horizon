# Cream by Solstar カートページマニュアル

オンラインストアの「カート」テンプレートで利用する専用セクションとブロックの実装ガイドです。ドロワーカートの挙動は別途`cart-drawer`セクションで管理されるため、本書ではページ型カートに特化しています。

## まずはここから（基本操作）
- テンプレート切替: テーマエディタ上部で「カート」を選択。プレビューURLが`/cart`に切り替わるのを確認。
- テストデータ: テスト購入用の商品をカートに追加してからエディタを開くと、商品行やサマリーが表示され設定が分かりやすい。
- ブロック整理: `main-cart`セクション内でタイトル/商品リスト/サマリーの順序を固定しつつ、追加ブロック（テキストや動画など）は「続きのブロック」領域に追加して告知を差し込む。
- 空カート確認: Cartが空の状態でプレビューすると、タイトルのみ中央寄せで表示され、追加ブロックは下部にまとまります。空状態の文言は翻訳ファイル(`locales/*.json`)で管理。

## セクション共通の見た目調整
- ここを触る: `main-cart`セクションの設定パネル。
- 主な項目: セクション幅（ページ幅/全幅）、カラースキーム、上下余白。
- ヒント: カラースキームをサマリーと同じものにすると余白のつなぎ目が自然になります。ヒーローの直後に配置する場合は上余白を0-16pxに詰めてください。

## テンプレート

### カート（main-cart）
- 目的: カートタイトル、商品リスト、注文サマリー、任意の追加ブロックを1つのセクションで管理。
- 含まれるブロック: `_cart-title`、`_cart-products`、`_cart-summary`（固定）＋テキスト/画像/ボタン/グループ/アプリ等の追加ブロック。
- 使う場面: 既定のカートページ。追加ブロックにクーポン説明、FAQ、配送案内、決済ロゴ等を配置してコンバージョンを補強。

設定詳細（セクション）
| 設定 | 種別 | 説明 |
| --- | --- | --- |
| セクション幅 | Select | `page-width`で中央寄せ、`full-width`で背景を端まで広げる。サマリーの固定幅を保ちたい場合はpage推奨。 |
| カラースキーム | Color scheme | 背景・テキスト・リンク色。サマリーブロックで別スキームを使う場合でも、外枠色はこちらが基準。 |
| 上下余白 | Range(px) | セクション外周。フッター直上では下余白32px程度確保すると呼吸感が出る。 |

ブロック: カートタイトル（_cart-title）
- `title`: 任意のテキスト。翻訳キー使用も可。
- `show_count`: 商品点数をタイトル横に表示（例: "Cart (3)"）。
- `type_preset` / `alignment`: フォントプリセットと左右寄せ。

ブロック: 商品リスト（_cart-products）
| 設定 | 説明 |
| --- | --- |
| ギャップ | 商品行の上下余白。デフォルト24px。 |
| 画像比率 | `portrait / square / landscape / adapt`。商品画像を統一比率で表示。 |
| 罫線（dividers） | 各行の境界線の有無。ONでリスト感、OFFでカード風に。 |
| ベンダー表示 | Vendor名をサブテキストに出すかを切替。 |

その他仕様
- 画像横にオプション/メタフィールド/プロパティ、ギフトカードフィールド等が自動表示されます。
- 各商品の数量調整、バリエーション表示、削除ボタン、メモ/ギフトラッピング等は`assets/component-cart-items.js`側と連動しています。

ブロック: サマリー（_cart-summary）
| 設定 | 説明 |
| --- | --- |
| extend_summary | ONでサマリーの背景ブロックをページ幅まで伸ばし、右カラムを大きく見せる。 |
| カラースキーム | 個別にサマリー背景/文字色を設定可能。`inherit_color_scheme`をONにするとセクション設定を継承。 |
| 枠線/角丸 | カード型にしたい場合に指定。 |

サマリー表示内容
- 小計、送料計算リンク、分割払い/Shop Pay表示、ディスカウントコード、注記欄、チェックアウトボタンなど`cart-summary`スニペットに準拠。
- カートメモや割引欄の有効化は「テーマ設定 > カート」で一括管理され、ここへ反映されます。

追加ブロック（自由エリア）
- `text / icon / image / button / video / group / spacer`などを使い、サマリー下に任意のコンテンツを配置。
- 例: 配送ポリシー、決済ロゴ、FAQ Accordions、マーケメッセージ。
- ブロック幅は`group`でラップし、PCは横並び/モバイルは縦並びに切替えるとレイアウトが安定。

### おすすめ商品（product-list セクション）
- 目的: カートページ下部で関連商品を提示し、セット買いを促す。
- 状態: テンプレートにプリセットとして含まれていますが初期は`disabled`。必要に応じて有効化してコレクションや表示件数を設定してください。
- 主な設定: コレクション選択、レイアウト（グリッド/カルーセル/エディトリアル）、表示上限、列数、ナビゲーションアイコン、上下余白。詳細は「コレクションリスト/商品リスト」マニュアルを参照。

## 連携・注意事項
- カートの入力項目（メモ欄・クーポン・分割払い表示など）は「テーマ設定 > カート」で集中管理。ページマニュアルと合わせて確認。
- サマリーブロックの色や角丸は、ギフトラップやアプリブロックが追加された場合にも適用されます。背景に半透明を使う場合、`extend_summary`をONにして高さを確保してください。
- Cartページとドロワーカートは同じJavaScriptロジックを共有します。片方だけをCSSで非表示にするより、テーマ設定で挙動を切り替える方が安全です。

## Tips（運用のコツ）
- カラーリング: サマリーを淡色、商品リストを白背景にすると、支払い部分への視線が自然に誘導されます。
- 信頼性強化: 追加ブロックに「配送目安」「返品ポリシー」「セキュリティバッジ」を入れると離脱抑止に効果的。
- クーポン表示: クーポンコードを案内する場合はテキストブロックをサマリー直下に設置し、`wrap: pretty`で折り返しを安定させる。
- 空状態CTA: 空カート時に誘導したいページ（例: 新着コレクション）へのボタンを追加ブロックで設置しておくと回遊が途切れません。

## 公開前チェックリスト
- カラースキーム: タイトル/商品行/サマリーで文字色・リンク色が十分なコントラストか。
- メモ/割引欄: テーマ設定でONにした項目が実際に表示され、フォーム送信時に正しく渡るかテスト。
- モバイル: 商品行のボタンや数量調整がタップしやすいか、サマリーが縦積みで読みやすいかを確認。
- 空カート: 翻訳文言やCTAが想定どおり表示されるか。余白が過度に大きくなっていないか。
- レコメンド: `product-list`セクションを有効化した場合、在庫切れ/非公開商品が含まれていないか、カルーセルが滑らかに動くか。
